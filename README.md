🌿 HabitFlow – Personal Wellness & Habit Tracker
-----------------------------------------------------------------------------------------
HabitFlow is a lightweight Android wellness application designed to help users build positive habits, monitor their daily mood, and stay consistent with hydration goals. The app provides a simple and distraction-free experience while storing all data locally for fast and fully offline performance.

HabitFlow focuses on promoting self-growth, emotional awareness, and healthy routines through an intuitive and user-friendly interface.

 Core Features
----------------------------------------------------------------------------------------
🧠 Mood Analytics & Emotional Insights

HabitFlow enables users to record their daily emotional states through a structured mood logging system.
The app helps identify emotional patterns over time, promoting self-awareness and encouraging mindful well-being through consistent tracking.

💧 Intelligent Hydration Management

The application includes a customizable hydration reminder system that helps users maintain optimal daily water intake.
Users can set personalized hydration goals and receive lightweight local notifications, ensuring consistent progress without impacting device performance.

✅ Structured Habit Management System

HabitFlow allows users to create, manage, and track personal habits with a streamlined daily progress interface.
The system encourages consistency by enabling simple completion tracking, helping users build sustainable routines over time.

💾 Offline-First Architecture

Designed with an offline-first approach, HabitFlow stores all user data locally using SharedPreferences.
This ensures:

Fast data access and smooth performance

Complete privacy and data security

No dependency on internet connectivity or cloud services

The lightweight local storage implementation guarantees a responsive and uninterrupted user experience.

🛠 Tech Stack
--------------------------------------------------------------------------------------
Language: Kotlin

UI Design: XML

IDE: Android Studio

Data Storage: SharedPreferences

Build System: Gradle (KTS)
